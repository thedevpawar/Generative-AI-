## 📊 Project: Parameter Estimation for a Probability Distribution

### 📌 Description
This project focuses on implementing the **Maximum Likelihood Estimation (MLE)** technique to estimate parameters of a probability distribution using a dataset.

---

### ⚙️ Tasks Performed

1. **Selection of Distribution**
   - Chose a probability distribution (e.g., Normal or Exponential).
   - Defined its PDF/PMF.

2. **Synthetic Data Generation**
   - Generated dataset using known parameters.

3. **MLE Implementation**
   - Implemented MLE algorithm in Python.
   - Estimated parameters from generated data.

4. **Performance Comparison**
   - Compared estimated parameters with true values.
   - Calculated estimation error.

5. **Analysis with Varying Sample Sizes**
   - Repeated experiments with different dataset sizes.
   - Studied how sample size affects estimation accuracy.
